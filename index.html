Installation

In this section, you can install YObject.

Firstly, you need to download this mod.

Download

Then you need to put this mod in Geode mods folder. This is example path to this folder

Path 
C:\Program Files (x86)\Steam\steamapps\common\Geometry Dash\geode\mods

That's all. You can open Geometry Dash and mod will be there
